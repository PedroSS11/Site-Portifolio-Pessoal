Mata Mosquito

Sobre o Projeto

O game Mata mosquito é um mini game desenvolvido em JavaScript, onde o objeitivo é eliminar todos os mosquitos dentro do tempo determinado. Um game bem simples mas onde estão em seu desenvolvimento, vários dos fundamentos da linguagem JavaScript. Um bom exercício de prática!
